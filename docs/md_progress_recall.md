## 技术架构

### 共享组件
- **认证**: Supabase Auth (同一账号双角色)
- **数据库**: Supabase PostgreSQL
- **AI服务**: 阿里云通义千问
- **部署**: Vercel

### 独立组件
| 组件 | Socrates | Essay |
|------|----------|-------|
| 框架 | Next.js 15 | Vite + React |
| 部署 | Vercel | Vercel |
| 仓库 | socra-platform (monorepo) |
