## 四、推广策略

### 1. 种子用户获取 (前100名)

```
策略: 精准定位 + 价值交换

渠道:
├─ 家长微信群 (小学/初中)
│   └─ 策略: 先分享干货，再软性推荐
│
├─ 教育类知乎回答
│   └─ 策略: 回答"如何辅导孩子"类问题
│
├─ 小红书种草
│   └─ 策略: "发现一个神仙学习工具"笔记
│
└─ 朋友圈裂变
    └─ 策略: 邀请3人免费体验
```

### 2. 内容引流 (持续)

```
高转化内容模板:

标题公式: 【数字】+ 【痛点】+ 【解决方案】
示例: "3个步骤，让孩子从被动学到主动学"

结构:
├─ 开头: 直击痛点，引发共鸣
├─ 中间: 分析原因，给出方法
├─ 结尾: 软性植入产品，引导尝试
└─ CTA: 扫码免费体验/加入交流群
```

### 3. 用户裂变

```
裂变机制:

1. 邀请奖励
   - 邀请1人 → 解锁高级功能7天
   - 邀请3人 → 免费体验1个月
   - 邀请10人 → 永久会员

2. 打卡分享
   - 连续学习7天 → 生成学习报告海报
   - 分享朋友圈 → 获得额外AI问答次数

3. 作业晒图
   - 上传错题分析结果 → 参与周排名
   - 前三名 → 获得奖励
```
