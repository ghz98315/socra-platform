# Socrates 几何下一轮优先级计划

- 日期：2026-04-13
- 当前阶段：坐标几何复合题基线已固化
- 目标：确定下一轮几何专项优化的最小范围，避免继续发散

## 当前已稳定

已通过或已固化回归基线：
- 非几何题误进几何链的门禁收紧
- 几何识别失败时保留上一版图形
- 低置信度 / unknown / error 状态的可操作化
- 坐标系中“反比例函数 + 几何组合”题型的 `C` 点恢复与缺线补全
- Geometry Wave 1 基本测试通过，当前可视为稳定基线

## 下一阶段优先级

建议转为“有样例再修 + 其他模块收口”的方式继续：

### P1：几何问题改为带样例再修

原则：
- 不再做纯猜测式优化
- 用户一旦给出新题图 / 新 OCR / 新失败截图，再按样例最小修复
- 已通过的 Wave 1 题型先保持冻结

### P2：回到待确认的提示词优化项

说明：
- 这部分修改必须先经用户确认
- 当前更适合先整理成可确认项，再进入实现

### P3：做一轮产品收尾与统一回归

重点：
- 把已通过链路沉淀为回归基线
- 把 remaining polish 项按优先级压缩
- 减少继续分叉开发

## 当前不建议做的事

当前不建议：
- 继续在几何 Wave 1 上做无样例微调
- 不经确认直接改主提示词
- 同时并行扩多个新功能面

## 建议的下一步执行方式

下一步建议直接进入：

1. 先将几何 Wave 1 视为当前稳定版本
2. 若再出现几何问题，要求附样例后再修
3. 将开发重心切到提示词待确认项与产品收尾
4. 新一轮改动前，仍以现有回归基线作为门槛
