# Socrates 几何下一轮优先级计划

- 日期：2026-04-13
- 当前阶段：坐标几何复合题基线已固化
- 目标：确定下一轮几何专项优化的最小范围，避免继续发散

## 当前已稳定

已通过或已固化回归基线：
- 非几何题误进几何链的门禁收紧
- 几何识别失败时保留上一版图形
- 低置信度 / unknown / error 状态的可操作化
- 坐标系中“反比例函数 + 几何组合”题型的 `C` 点恢复与缺线补全

## 下一轮优先级

建议按这个顺序继续：

### P1：圆 + 切线 / 直径题

优先原因：
- 这类题在初高中几何里频率高
- 常见错误是：
  - 切线没画出来
  - 直径画成普通弦
  - 切点与圆心半径关系不完整

最小验收点：
- `circleO` 正常渲染
- 切线 `CD` 能显示
- 半径 `OC` 能显示
- `∠OCD=90°` 关系不丢

### P2：矩形 / 投影 / 垂线交点题

优先原因：
- 已有专项 fallback
- 但这类题对 `E/F` 点、垂足、交点位置要求高

最小验收点：
- `E`、`F` 点稳定出现
- `BE`、`CF` 能正常显示
- `F` 不再漂到错误位置

### P3：函数图像 + 辅助线题

优先原因：
- 当前已解决一个高频样例
- 但仍可能存在：
  - 曲线经过点不完整
  - 与坐标轴或已知线段交点未绑定
  - AI 给了点但没给交线

最小验收点：
- 曲线与指定点绑定稳定
- 关键辅助线可补出
- 曲线与几何图不再分裂成两套独立图层

## 当前不建议做的事

本轮暂不建议：
- 再扩主提示词
- 再改对话链
- 做大规模几何系统重构

原因：
- 当前最值钱的是继续压高频误判和漏画问题
- 大改提示词会把已通过的链路重新带入不稳定区
