## 一、商业化功能测试 (重点)

### 1.1 订阅页面 `/subscription`

**入口方式:**
- 直接访问: https://socrates.socra.cn/subscription
- Dashboard 页面点击 Pro 学科（物理/化学）的「开通 Pro」按钮

**测试项目:**

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 页面加载 | 访问 /subscription | 显示三个套餐卡片（月度/季度/年度） |
| 套餐切换 | 点击不同套餐 | 高亮选中，底部价格更新 |
| 当前状态 | 已登录用户 | 显示当前会员状态和到期时间 |
| 优惠码-有效 | 输入有效优惠码 | 显示折扣价格 |
| 优惠码-无效 | 输入无效优惠码 | 提示"优惠码无效" |
| 订阅按钮 | 点击底部订阅按钮 | 跳转到 /payment 页面 |
| 权益对比 | 查看功能对比区域 | 显示 Pro vs Standard 权益差异 |

**套餐价格:**
- 月度会员: ¥29.9/月
- 季度会员: ¥79.9/季 (热门)
- 年度会员: ¥239.9/年 (最划算)

---

### 1.2 支付页面 `/payment`

**入口方式:**
- 从订阅页面点击「立即订阅」
- 直接访问: https://socrates.socra.cn/payment

**测试项目:**

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 套餐选择 | 切换月度/季度/年度 | 价格和折扣实时更新 |
| 支付方式 | 选择微信/支付宝 | 高亮选中支付方式 |
| 优惠码 | 输入优惠码 | 价格更新，显示折扣金额 |
| 订单汇总 | 查看底部汇总 | 显示原价、折扣、实付金额 |
| 返回修改 | 点击返回 | 返回订阅页面 |
| 确认支付 | 点击确认支付 | 调用支付API，跳转支付成功页 |

**支付流程:**
```
选择套餐 → 选择支付方式 → (可选)输入优惠码 → 确认支付 → 支付成功页
```

**入口方式:**
- 支付完成后自动跳转

| 返回首页 | 点击「返回首页」 | 跳转到 /dashboard |
