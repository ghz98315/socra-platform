# Socrates 几何专项人工回归与部署前清单

- 日期：2026-04-13
- 适用范围：本轮几何恢复收口 + 几何门禁收紧
- 目标：验证“真几何题能进几何链，非几何题不误进；识别不稳时不丢当前图形”

## 一、执行前准备

### 本地命令入口

在仓库根目录执行：

```powershell
pnpm.cmd socrates:status:local
```

如果未启动，再执行：

```powershell
pnpm.cmd socrates:start:local
```

如果需要重新启动：

```powershell
pnpm.cmd socrates:stop:local
pnpm.cmd socrates:start:local
```

### 类型检查

```powershell
$env:NODE_OPTIONS='--max-old-space-size=4096'
pnpm.cmd --filter @socra/socrates exec tsc --noEmit
```

说明：
- 当前工程在默认 Node 堆下可能会触发 `heap out of memory`
- 加上 `NODE_OPTIONS` 后本轮已通过

## 二、人工回归主清单

### Case 1：无图普通数学题，不应误进几何链

可直接粘贴 OCR 文本示例：

```text
已知 2x + 3 = 11，求 x 的值，并说明解方程的步骤。
```

预期：
- 不出现几何画板
- 若出现几何提示卡，也应是“未稳定识别出几何图形 / 继续答题”，不能生成猜测图形
- 聊天区可直接继续解题

### Case 2：普通应用题，含少量字母，不应误进几何链

```text
甲、乙两人同时从 A 地出发前往 B 地，甲每小时行 5 千米，乙每小时行 4 千米，3 小时后两人相距多少千米？
```

预期：
- 不出现几何画板
- 不能因为出现 `A 地`、`B 地` 就误识别成线段 AB

### Case 3：函数图像题，应进入几何链

```text
如图，反比例函数 y=3/x 的图像经过点 A(1,3)，与 x 轴交于点 B，求相关结论。
```

预期：
- 可进入几何/图像链
- 画板或图像区域可显示
- 若置信度不高，允许显示“待确认”，但不能被误拦截

### Case 4：标准三角形题，应进入几何链

```text
如图，在△ABC中，AB=AC，∠A=40°，D 为 BC 上一点，求 ∠BDC。
```

预期：
- 出现几何画板
- 识别结果为三角形或可用复合图
- 若低置信度，显示黄提示条，但仍可继续答题

### Case 5：圆题，应进入几何链

```text
如图，AB 是 ⊙O 的直径，点 C 在圆上，连接 AC、BC，求证 ∠ACB=90°。
```

预期：
- 出现几何画板
- 识别为圆或圆相关图形
- 不应被门禁拦掉

### Case 6：先成功识别，再把 OCR 改坏，旧图应保留

步骤：
1. 先用 Case 4 或 Case 5 成功识别出图形
2. 再把 OCR 文本改成缺少关键条件的版本，例如删掉“如图”“△ABC”“∠A=40°”
3. 点击“重新识别”

预期：
- 原有几何画板仍保留
- 出现“本次未稳定识别出新图形，已保留上一版图形”或“本次几何识别异常，已保留当前图形”的提示
- 已添加的辅助线、当前解题进度不被冲掉

### Case 7：低置信度场景，仍可继续答题

建议做法：
- 输入一个图形描述不完整的题，例如只有部分点和关系

示例：

```text
如图，A、B、C、D 为四边形顶点，已知 AB∥CD，点 E 在 AD 上，求证相关结论。
```

预期：
- 可以出现画板
- 画板顶部显示“待确认”或黄提示条
- 用户仍可继续对话，不会卡死在几何区域

### Case 8：切换到非数学学科，不应出现几何链

步骤：
1. 切到语文或英语
2. 粘贴任意带字母的文本

预期：
- 不出现几何画板
- `/api/geometry` 不应再成为主要路径

### Case 9：坐标系中的反比例函数 + 几何组合题，C 点与线段必须完整

建议优先使用原题图：

```text
D:\github\Socrates_ analysis\test_image\PixPin_2026-02-28_07-35-38.png
```

可参考 OCR 文本：

```text
如图，Rt△OAB与Rt△OBC位于平面直角坐标系中，∠AOB=∠BOC=30°，BA⊥OA，CB⊥OB，若AB=√3，反比例函数y=k/x恰好经过点C，则k=______。
```

预期：
- `C` 点必须识别出来
- `BC`、`OC` 等关键线段必须能画出来
- 反比例函数曲线必须经过 `C`
- 不能退化成只有函数曲线和 `A/B/O` 点

当前状态：
- 已人工复测通过

## 三、重点观察项

每个 case 建议重点看这几项：

- 是否误出现几何画板
- 是否误生成猜测图形
- 低置信度时是否给出“待确认/核对条件”提示
- 识别失败时是否保留上一版图形
- 能否直接继续答题，不被几何链卡住
- 数学题与非数学题的分流是否清晰

## 四、部署前最小通过标准

满足以下条件即可进入部署：

- 类型检查通过
- Case 1 通过
- Case 3 或 Case 4 至少通过一个
- Case 6 通过
- Case 8 通过

如果时间充足，建议 8 个 case 全过再推送正式部署。

## 五、部署后快速抽检

部署完成后至少抽检 3 项：

- 无图题不再误弹几何卡
- 三角形题仍能识别
- 成功识别后再次失败时旧图仍保留
